## 🚀 Descripción del Proyecto
La aplicación resuelve el descontrol administrativo en el Colegio Educativo Molaco, automatizando procesos que anteriormente se realizaban de forma manual para aproximadamente 60-80 estudiantes.

### Funcionalidades Clave:
* **Módulo de Estudiantes:** Registro y gestión centralizada del alumnado.
* **Control de Asistencia:** Registro diario con alertas visuales para ausencias frecuentes.
* **Gestión de Pagos:** Seguimiento de cobros mensuales y estados de cuenta (pagado, parcial, pendiente).
* **Dashboard Administrativo:** Panel con métricas de ingresos estimados, pagos pendientes y estudiantes activos.
* **Módulo de Pendientes:** Área para gestionar tareas administrativas por hacer.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** React + Vite.
* **Backend / Database:** Supabase.
* **Despliegue:** Netlify.
